# CLAUDE.md - Project Instructions

## Project: Drift

An anti-engagement app that restores boredom and cognitive friction.

## Quick Reference

- **Manifesto**: `freedom-requires-friction-manifesto.md` - the philosophical foundation
- **Dev Guide**: `AGENTS.md` - detailed development principles and guidelines

## Core Commands

```bash
# Development (once set up)
npm run dev      # Start dev server
npm run build    # Production build
```

## Key Principles (Quick Check)

Before any change, verify:

1. Does NOT increase engagement
2. Does NOT add gamification
3. Keeps the app forgettable
4. Maintains intentional friction
5. Stays minimal

## Tech Decisions

- Vanilla/minimal JS (no heavy frameworks)
- Local-first (no user accounts unless essential)
- No analytics beyond basic function
- Muted, calm visual design

## What Success Looks Like

Users eventually stop needing the app. Declining usage = victory.
